# 关于

## 个人介绍

一个程序员，目前专注于Java Web开发。

## 关于本站

这个博客是我用来分享自己学习或工作上的心得和体会，记录一些自己感兴趣的技术的地方，希望也能给你提供一些帮助。

### 本站主要用到的技术及主要支持的特性

#### Blog System

* Jekyll，本站使用的博客引擎，[详情](https://jekyllrb.com)
* NexT，本站使用的博客主题，[详情](https://github.com/simpleyyt/jekyll-theme-next)

#### Deployment

* Debian 9

#### Web Server

* Nginx 1.14.x
* OpenSSL 1.1.1， 启用TLSv1.3，[详情](https://www.openssl.org/blog/blog/2018/09/11/release111/)

#### HTTPS & HTTP/2

* 支持 ECC 证书，[详情](https://blog.njilc.com/about)
* 支持 TLS v1.3，[详情](https://blog.njilc.com/about)
* 支持 HTTP/2，[详情](https://blog.njilc.com/about)

#### Security

* ~~CSP3，Content Security Policy Level 3，[详情](https://developer.mozilla.org/en-US/docs/Web/HTTP/CSP)~~
* Security Response Header，使用 HTTP 响应头部增强网站安全，[详情](https://infosec.mozilla.org/guidelines/web_security.html#cross-origin-resource-sharing)

#### Experience

* Mobile friendly，
* Markdown，本站文章使用 Markdown 格式编写，[详情](https://guides.github.com/features/mastering-markdown/)

#### Third-Party Service

* Let's Encrypt， 本站使用的证书提供商，[详情](https://letsencrypt.org)
* GitHub Pages， 本站使用的站点托管服务，[详情](https://pages.github.com)
* Cloudflare， 本站使用的CDN提供商，[详情](https://www.cloudflare.com)
